jQuery 对象的方法:
1. text(): 
	1). 若没有参数, 则获取指定 jQuery 对象的文本子节点的值
	2). 若有参数, 则设置指定 jQuery 对象的文本子节点的值
	
2. each():
	1). 对获取到的 jQuery 对象(DOM 对象的数组)进行遍历	, 但需要注意在进行遍历时, 遍历的对象为 DOM 对象, 而非 jQuery 对象
	
3. val(): 
	1). 若没有参数, 则获取指定表单对象的 value 属性值
	2). 若有参数, 则设置指定表单对象的 value 属性值
	
4. attr(): 类似于 1, 3, 只不过用于获取或设置属性值
